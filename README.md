# TELOS (Twitter-like Sharable Vocabulary List App)
__How to run TELOS on your computer__
```
$ rails server
```
Then, access http://localhost:3000

## Things I'm currently working on
* Search feature
  * Search Users
  * Search Posts
  * Search Tags
* My Vocab List
* User/Tag deletion
* Improve UX
